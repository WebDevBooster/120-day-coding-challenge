CSS Flexbox:
Responsive Forms

Resize the browser window to see the effects on the forms and form fields below.

The following long registration form is designed for demonstating the capabilities of CSS flexbox to adjust the form fields and layout without the need to rely on media queries etc. It makes the styling of forms easier and faster. In real world applications you should avoid having 2-column form layouts because they are not user-friendly. So, this form is for flexbox demonstration only and the HTML for this form is very semantic. No extraneous markup!
